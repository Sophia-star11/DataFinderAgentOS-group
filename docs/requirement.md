# DataFinderAgentOS 需求文档

## 系统背景
- **系统名称**：某政务智能瞭望与智能问数系统
- **系统类型**：Web B/S 结构的数据智能采集和数据智能问数系统
- **架构划分**：用户侧-前台 + 管理侧-后台
- **用户侧定位**：类 ChatGPT 的智能应用，用户通过 Chat 模式与数据直接对话，实现智能问数，通过模型直接获得数据统计、分析、生成图形图像报表
- **用户侧功能**：用户登录、用户注册、用户对话、数字员工、@天气、@新闻、@音乐、@电影、@yummy、报表功能、历史记录、报告导出
- **管理侧功能**：用户管理、功能管理、菜单管理、角色管理、瞭望采集、瞭源管理、数据仓库、深度采集、数字员工、模型引擎、数智大屏、舆情大屏

## 文档定位
- 本文件用于跟踪项目需求及完成状态，反映当前仓库的真实实现情况。
- 需求以用户侧-前台和管理侧-后台的模块划分为依据，标注完成状态。

---

## 一、已完成需求

### 1. 基础框架与认证
- **用户登录/退出** - 支持前后台双通道登录（`/login` 和 `/admin/`），基于 secure_cookie 会话维持
- **首页仪表盘** - 后台首页展示登录欢迎信息和基础导航
- **RBAC权限模型** - 用户-角色-功能三级权限体系，角色功能分配 + 菜单自动同步

### 2. 控制台图表化（管理侧）
- ECharts 替换纯数字统计卡片，30秒自动轮询
- 深度采集覆盖率环形图（`/api/dashboard/stats` 返回 `deep_coverage`）
- 近7天数据入库趋势折线图（`daily_trend`）
- 采集任务状态分布柱状图（`task_status_bar`，状态映射：待处理/运行中/已完成/失败）
- `static/js/dashboard-charts.js` 封装图表渲染模块

### 3. 用户管理（管理侧）
- 用户 CRUD（创建/读取/更新/删除）
- 分页搜索、状态管理
- 超管保护机制（不可修改/删除超管账号）
- 密码编辑（支持在用户表单中重置密码）
- 管理员标识（admin角色不可删除）

### 4. 功能管理（管理侧）
- 功能 CRUD（支持父子层级树结构）
- 启用/禁用切换
- 级别展示（通过图标和层级缩进显示功能级别）

### 5. 菜单管理（管理侧）
- 菜单 CRUD（按角色配置菜单）
- 排序管理
- 菜单预览
- 自动同步：角色功能分配变更时自动更新菜单
- **图标图形化展示**：菜单列表中功能图标以图形图标形式展示

### 6. 角色管理（管理侧）
- 角色 CRUD
- 角色-功能分配（支持多选树）
- 管理员保护
- 菜单自动生成
- **编辑弹窗**：完整实现编辑弹窗功能（AJAX获取角色数据→弹窗回填→保存更新），支持状态修改

### 7. 瞭望采集（管理侧）
- **瞭源管理（采集管理）** - 采集源CRUD，支持配置URL模板、请求头、分页参数
- **瞭望采集执行** - 按关键词+分页规则批量爬取
- **采集结果橱窗展示** - 卡片式布局，支持全选/全不选、批量保存到数据仓库
- **每页步长可配置** - B区步长可编辑并同步瞭源配置
- **分页采集** - 支持手动选择采集页数

### 8. 数据仓库（管理侧）
- **数据仓库管理** - 列表/详情/删除/批量删除
- **采集数据存储** - 瞭望采集结果保存到数据仓库（支持去重upsert）
- **深度采集状态标记** - 列表中标识是否已深度采集
- **左侧导航菜单集成**
- **批量操作** - 支持批量选中后删除/深度采集

### 9. 瞭源管理（管理侧）
- **采集源规则配置** - URL模板、请求头、分页参数、关键词参数配置
- **采集源CRUD** - 支持创建/编辑/删除采集源

### 10. 模型引擎（管理侧）
- 模型 CRUD（支持不同API提供商配置）
- 模型分类管理（text/image/audio/video/multimodal/embedding）
- 默认模型设置
- SSE流式AI对话（代理到OpenAI兼容API）
- Token统计使用量
- 对话参数调节（Temperature/Top-P/Max Tokens）
- Markdown渲染（marked.js库）

### 11. 数字员工（管理侧）
- 数字员工 CRUD
- 支持两种类型：LLM型（关联模型+系统提示词+技能）和 API型（自定义HTTP端点）
- 关联模型选择
- 数字员工测试功能（LLM型调用关联模型对话，API型发送HTTP请求验证）
- 智能中枢菜单组织（模型引擎+数字员工归入统一父菜单）

### 12. 深度采集（数据仓库子功能）
- 单条数据深度采集（通过数字员工-采集专员执行）
- 批量深度采集（队列模式，所有任务独立监控）
- crawl4ai 网页爬虫集成（爬取页面正文作为分析上下文）
- 任务监控面板（悬浮窗，实时显示进度/步骤/日志/员工详情）
- 采集结果持久化（deep_collect_data 表存储网页爬取内容和AI分析结果）
- 更新采集（覆盖旧采集数据）
- 已采集数据查看结果

### 13. 个人信息管理
- 查看个人信息
- 修改当前用户密码

### 14. 用户注册（用户侧）
- 注册页面 `register.html`（双栏品牌布局）
- 注册API `/api/user/register`（含用户名长度、密码长度、重复密码校验）
- 注册后自动分配普通用户角色（role_id=2）
- 用户名唯一性检查

### 15. 用户对话（用户侧）
- **chat.html** 五区布局（A区LOGO+标题 / B区模型切换 / C区历史列表 / D区对话 / E区输入）
- SSE流式对话输出（通过 `UserChatApiHandler` 代理到OpenAI兼容API）
- Markdown渲染（通过 markdown-it 库）
- 聊天气泡样式（左AI右用户）
- 对话历史本地存储（localStorage）
- 对话任务列表（C区，支持切换/删除/新建对话）
- 欢迎页面和快捷建议
- 响应时间与Token用量显示
- 意图识别与数据查询（支持与数据库对话获取数据反馈）

### 16. 用户侧数字员工与@技能命令
- 对话界面 `@` 菜单激活数字员工列表
- 支持 `@天气`、`@新闻`、`@音乐`、`@电影`、`@yummy` 等技能调用
- LLM型数字员工优先使用关联模型，未配置则使用默认模型
- API型数字员工直接调用配置的HTTP端点
- 消息气泡显示当前激活的数字员工名称标签

### 17. 用户侧快捷功能
- 对话界面 `/` 菜单激活快捷功能（生成报表、生成图表、数据分析、导出报告）

### 18. 控制台实时刷新
- 8项统计指标 + 3个ECharts图表，每30秒自动轮询更新
- 接口 `/api/dashboard/stats` 返回统计、趋势、分布数据

### 19. 数智大屏（管理侧）
- 数据可视化大屏（`admin/data_screen.html`）
- 桑基图（Sankey）数据流水线（瞭望源→数据仓库→深度采集→任务状态，真实数据驱动）
- 关键词词云（echarts-wordcloud）
- 数据来源分布饼图
- 近30天数据趋势折线图
- 与管理后台统一布局（左侧导航栏 + 顶栏 + 底部）
- 30秒数据自动轮询刷新
- 独立API：`/api/data-screen/stats`、`/api/data-screen/wordcloud`、`/api/data-screen/trends`、`/api/data-screen/source`
- 集成在智能中枢菜单下（function code: `data_screen`）

### 20. 其它
- 用户侧登录/注册样式统一
- Logo标题显示修复
- 智能中枢图标更新（layui-icon-bot）
